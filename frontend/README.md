# Accelrate Frontend

## Quick Start

To keep your secret key safety, you should not push any environment files into repo.

Create `.env.local` file in `frontend/` and add these keys:

```
VITE_API_URL="http://localhost:3000/"
```

### 2. Install dependencies

```sh
bun install
```

### 3. Start

```sh
bun run dev
```
